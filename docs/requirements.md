# 用户需求规格

## 项目名称

Windows 历史剪贴板工具

## 用户画像

- 编程小白，日常办公使用 Windows 电脑
- 经常需要复制粘贴，但系统剪贴板只能记住最后一条
- 希望能够回溯和复用之前的复制历史

## 功能需求

### F1 - 自动记录剪贴板内容

- 实时监控系统剪贴板变化
- 自动记录文字内容和图片内容
- 相同内容不重复记录（去重）
- 后台静默运行，不打扰用户正常操作

### F2 - 存储期限设置

- 提供预设选项：1天 / 3天 / 5天
- 超过期限的非置顶记录自动清理
- 置顶内容永不过期

### F3 - 历史记录浏览

- 以卡片列表形式展示
- 时间降序排列（最新的在最上面）
- 置顶内容始终排在最前
- 文字卡片：显示内容预览（截断至80字）
- 图片卡片：显示缩略图（120×80px）

### F4 - 记录操作

- 点击卡片：将内容复制回剪贴板
- 置顶/取消置顶：重要内容固定在列表顶部
- 删除：移除不再需要的记录
- 清空全部：一键清除所有历史

### F5 - 搜索功能

- 支持按文字内容搜索
- 实时过滤（输入即搜）

### F6 - 运行形态

- 系统托盘小工具（任务栏右下角图标）
- 左键点击图标打开/关闭面板
- 右键菜单：显示历史、开机自启、退出

### F7 - 全局热键

- Ctrl+Shift+V：弹出/收起历史面板

### F8 - 开机自启动

- 可选开启，开启后在 Windows 启动文件夹创建快捷方式

## 非功能需求

### NF1 - UI 设计
- 简洁、直观、干净
- 主色调：淡蓝色
- 窗口尺寸适中，不占用过多屏幕空间

### NF2 - 数据安全
- 数据仅本地存储，不上传网络
- 不记录密码等敏感信息（暂不实现，后续可加）

### NF3 - 性能
- 内存占用低
- 剪贴板轮询不影响系统性能
- 面板打开响应迅速
