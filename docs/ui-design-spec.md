# UI 设计规范

## 设计原则

- **简洁**：减少视觉噪音，每个元素都有其存在理由
- **直观**：操作无需学习，一看就懂
- **轻量**：窗口小巧，不抢占桌面空间
- **一致性**：所有卡片、按钮、交互行为统一

## 卡片设计

### 文字卡片

```
┌──────────────────────────────────────┐
│ 📝 这是一段被复制的文字内容，如果超...│ ← 截断到 80 字
│                                      │
│ 2分钟前                     📌  🗑  │ ← 操作按钮 (hover 显示)
└──────────────────────────────────────┘
```

### 图片卡片

```
┌──────────────────────────────────────┐
│ ┌──────────┐                         │
│ │          │  图片 · 256KB           │ ← 左侧缩略图，右侧信息
│ │ 缩略图   │  5分钟前                │
│ │          │              📌  🗑     │
│ └──────────┘                         │
└──────────────────────────────────────┘
```

### 置顶卡片

```
┌──────────────────────────────────────┐ ← 背景色 #E8F4F8
│ 📌 已置顶的文字内容...               │ ← pin 标记始终可见
│ 1小时前                     📌  🗑  │
└──────────────────────────────────────┘
```

## 交互行为

| 操作 | 行为 |
|------|------|
| 单击卡片 | 内容复制到剪贴板 → 面板自动隐藏 → 托盘提示"已复制" |
| 鼠标悬停卡片 | 显示操作按钮（置顶/删除），卡片微微上浮阴影 |
| 双击卡片 | 同单击 |
| 搜索输入 | 实时过滤列表，无结果时显示"没有找到匹配的记录" |
| 存储天数切换 | 立即保存设置，触发清理 |
| 清空全部 | 弹出确认对话框，确认后删除所有非置顶记录 |
| 删除单条 | 直接删除，无确认弹窗（可撤销？暂不实现） |

## 窗口行为

- 失去焦点时自动隐藏（点击面板外部区域）
- 始终置顶（在所有窗口上方）
- 不显示在任务栏
- 打开位置：鼠标所在屏幕的右下角区域
- 动画：弹出时从托盘位置微微向上展开（可选）

## 图标

- 托盘图标：16×16px，淡蓝色剪贴板图标（程序化生成，无需外部图片文件）
- 置顶图标：📌 使用 Unicode 字符
- 删除图标：使用文字"删除"或简单 ✕ 符号
